# CV

### Installation

First step

```sh
$ git@github.com:Amstergo/cv.git
```

Then install the dependencies and devDependencies.

```sh
$ cd cv
$ yarn install
```

#### Run project

Start WDS (webpack-dev-server)

```sh
$ yarn run start:dev
```

Open in your browser

```sh
http://localhost:9000/
```
